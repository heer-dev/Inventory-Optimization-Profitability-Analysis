# 🧑‍💼 User Stories

### User Story 1

As a business owner, I want to identify loss-making products so that I can stop selling or reprice them.

### User Story 2

As an inventory manager, I want to identify slow-moving products so that I can reduce storage costs.

### User Story 3

As a sales manager, I want to identify top-performing products so that I can increase promotions.

### User Story 4

As a regional manager, I want to analyze region-wise performance so that I can improve weaker regions.

### User Story 5

As a finance analyst, I want to track profit margins so that I can ensure business sustainability.
